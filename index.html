How to play

  • Extract the zip file you download

  • Go to Minecraft folder

  • Select a version

  • Run the [version].py to open the game

  • version 2.2 , dev version and game.py require player name!

  • W A S D to move

  • Space to jump

  • Left clic to break block

  • Right click to place block

  • ESC to exit the game


  • You can also try dev versions

System requirements

  • Python 3

  • Ursina engine

  • CPU - intel core i9-12900 , 5.1 GHz or better

  • Graphic - nvidia RTX-3090 24 GB or better

  • OS - Windows 10 21H2 or higher
